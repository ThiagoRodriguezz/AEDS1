# 📚 AEDS I — Algoritmos e Estruturas de Dados I

Repositório com as atividades desenvolvidas na disciplina **Algoritmos e Estruturas de Dados I** do curso de Ciência da Computação da **PUC Minas**.

---

## 🗂️ Organização

```
AEDS1/
├── README.md
├── listas/
│   ├── lista01/
│   │   └── Exercicio.c
│   ├── lista02/    
└── └── ...
```

Cada pasta dentro de `listas/` corresponde a uma lista de exercícios da disciplina.


## ⚠️ Observações

A maioria dos exercícios utiliza a biblioteca `io.h`. Certifique-se de que seu compilador suporta essa lib antes de rodar os programas.

---

## 🛠️ Tecnologias

- **Linguagem:** C / C++
- **Compilador:** GCC / G++

---

## ▶️ Como compilar e rodar

### C
```bash
gcc -o programa exercicio01.c
./programa
```

### C++
```bash
g++ -o programa exercicio01.cpp
./programa
```
